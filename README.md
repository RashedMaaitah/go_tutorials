# Go Language Learning Repository

Welcome! This is a repository dedicated to learning the basics of the Go programming language.

## Purpose

This repo contains exercises, examples, and projects as I explore Go's fundamentals, including:

- Variables and data types
- Functions and methods
- Control flow
- Packages and imports
- Concurrency basics
- Error handling

## Getting Started

To run the code in this repository, ensure you have [Go installed](https://golang.org/doc/install).

```bash
go run <file.go>
```

Feel free to explore, experiment, and learn along the way!
